# Day 2 — CD4017 Sequencer

## Goal
Begin the implementation of a sequential LED system using the CD4017 decade counter driven by a NE555 timer.

## Activities
- Created the repository structure for the CD4017 module.
- Reviewed the datasheet and pin configuration of the CD4017.
- Planned the integration with the previously developed 555 timer circuit.

## Insights
- The CD4017 acts as a digital counter that advances with each clock pulse.
- Integrating analog and digital components is essential for building more complex systems.

## Next Steps
- Connect power and ground to the CD4017.
- Link the NE555 output to the clock input (pin 14).
- Test the first LED connected to output Q0.
